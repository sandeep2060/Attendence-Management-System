# AttendEase Tracker
**A simple AttendEase written using different JavaScript Libraries, [React](https://react.dev) and [Express](https://expressjs.com).**

---

## Installation

```bash
git clone --depth=1 https://github.com/sandeep2060/Attendence-Management-System.git
npm install # NOTE: first cd into frontend and backend directory
```

## Start the development server
```bash
npm run dev # For frontend
npm start # For backend
```
